# SAST, IAC and SCA by CybeDefend

CybeDefend integrates powerful scanning engines with leading industry tools to cover every aspect of application security:

## Infrastructure as Code (IAC)
- **Engine:** CybeDefend Engine  
- **Tools:** Checkov, Trivy, Kicks  
- **Focus:** Analyzing configuration files and infrastructure scripts for vulnerabilities.

## Static Application Security Testing (SAST)
- **Engine:** CybeDefend Engine  
- **Tools:** Semgrep, Bearer  
- **Focus:** Scanning your source code across various supported languages.

## Software Composition Analysis (SCA)
- **Engine:** CybeDefend Engine  
- **Tools:** GitHub Advisories integration  
- **Focus:** Identifying vulnerabilities in third-party dependencies.
