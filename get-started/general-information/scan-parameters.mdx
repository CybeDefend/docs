# Parameters to Start Your Scan Project

Tailor your scanning process with a variety of configurable parameters:

- **Scanning Frequency:** Define how often scans should occur.
- **Vulnerability Severity Choice:** Select which levels of vulnerability severity to report.
- **Type of Scanning:** Choose the recommended scanning method for optimal results.
- **SAST Fast Scanning:** Enable rapid static analysis for immediate feedback.
- **Dataflow Option:** Activate dataflow analysis to trace vulnerability propagation.
