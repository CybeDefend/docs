## Overview

The CybeDefend plugin for JetBrains IDEs provides comprehensive security analysis directly within your development environment. Identify and address vulnerabilities in your code, dependencies, and infrastructure configurations without leaving IntelliJ IDEA, PyCharm, WebStorm, or other JetBrains IDEs.

## Key Features

### Comprehensive Security Analysis

- **Code Security (SAST)**: Find vulnerabilities in your custom code
- **Dependencies Security (SCA)**: Detect known vulnerabilities in open-source libraries
- **Infrastructure Security (IaC)**: Uncover misconfigurations in infrastructure-as-code files

### In-IDE Results

View scan summaries and detailed vulnerability information directly within dedicated JetBrains IDE panels.

### Direct Code Navigation

Quickly jump from a vulnerability report to the exact line of code that needs attention.

---

## Getting Started

### 1. Install

Open **Settings/Preferences** → **Plugins** → **Marketplace**, search for "**CybeDefend**", and click **Install**.

### 2. Authenticate

After installation, configure your CybeDefend API key in the plugin settings.

### 3. Configure Project

Select the corresponding CybeDefend Organization and Project for your workspace.

### 4. Scan

Open the CybeDefend tool window and start a vulnerability scan with one click.
